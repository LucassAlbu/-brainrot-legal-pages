# Brainrot Local legal pages

Static GitHub Pages site for TikTok app review.

Required public URLs after publishing:

- Terms of Service: `https://YOUR_GITHUB_USERNAME.github.io/brainrot-legal-pages/terms.html`
- Privacy Policy: `https://YOUR_GITHUB_USERNAME.github.io/brainrot-legal-pages/privacy.html`

Publish with GitHub Pages:

1. Create a public GitHub repository named `brainrot-legal-pages`.
2. Upload all files from this folder to the repository root.
3. Go to repository Settings > Pages.
4. Set Source to `Deploy from a branch`.
5. Select branch `main` and folder `/(root)`.
6. Save and wait for GitHub Pages to publish.
